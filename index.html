.pyBot

A general moderation bot for Discord, developed in Python.

Features

Python

Like the name suggests, .pyBot is developed in Python, using the discord.py library, allowing the bot to be extremely fast and reponsive.

We won't slow your server down!

General Moderation

All of the bot's moderation actions are logged in your server's audit log - just like a normal punishment!

.pyBot currently has two moderation commands, --kick and --ban.

Moderation commands are all extremely easy to use, allowing you to quickly deal with people breaking your server's rules.

Commands

helpme: Lists commands and their specific parameters

math: Add, subtract or multiply any number

kick: Kick a specified user

ban: Ban a specified user

setchannel: Sets the pyBot's announcement channel

serverinfo: Lists information for the server

channelinfo: Lists information for the channel
